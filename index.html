What Is This App?
GitHub Connection Visualizer lets you explore, analyze, and understand your followers and followings without any login or personal access token.
  • See who follows you back, spot inactive connections, and decide who to keep or remove.
  • This is not a follow-unfollow script.
  • It is a secure, open-source tool that gives clear visual insights so you can connect with developers you value or clean up accounts you no longer need.
This app has a small learning curve. For the best experience, It is recommend reading the Usage Notes before you start.

How to use the app effectively:
  • If your GitHub account has under 6.0k (6,000) followings and 6.0k (6,000) followers, you can usually scan all data smoothly within about an hour.
  • For larger accounts, GitHub’s API rate limits can slow the process. To handle this, the app uses a Data Scanning Monitor and a Progress Tracking Mechanism that lets you gather data in safe, repeatable steps until the scan is complete.
  • You can also pause at any time by creating a backup of your current scan. Later, simply restore the backup to continue where you left off without starting over.
