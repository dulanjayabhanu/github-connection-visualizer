Privacy Policy
Effective Date: 2025-03-11 Last Updated: 2025-03-11

Your privacy is important to us. GitHub Connection Visualizer is designed to respect your data and keep your GitHub information safe. This application works entirely in your browser, meaning no personal data, passwords, or access tokens are ever stored on our servers. The purpose of this Privacy Policy is to explain what limited information we handle, how we use it, and how we keep it secure.

  1. Data We Collect
    • GitHub Usernames: Only the username you enter is used to request public data from the GitHub API.
    • Backup Data: You can export encrypted backups of your scan results. These files are created and stored only on your device.
    • No Personal Access Tokens: We never request or store GitHub passwords or access tokens.
  2. How We Use Data
    • Public GitHub API responses are processed locally in your browser to generate visualizations.
    • Backup files remain entirely under your control and are never uploaded to any server.
  3. Local Storage

    Your scan progress may be temporarily stored in your browser’s local storage to support backup and restore functionality.

    This data stays in your browser and is not shared with anyone.

  4. Third-Party Services

    The Application uses GitHub’s public API to retrieve data. Your use of GitHub remains subject to GitHub’s own Privacy Policy.

  5. Security

    All backup files are encrypted in your browser using a user-generated passphrase before download. We never have access to your passphrase or encrypted data.

  6. Changes to Policy

    We may update this Privacy Policy from time to time. Continued use of the Application means you accept the updated policy.

GitHub Connection Visualizer is not affiliated with or endorsed by GitHub. All content posted here is public and owned by its respective authors.
